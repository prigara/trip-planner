# Trip Planner

A drag-and-drop trip planning app for organizing day-by-day itineraries. Plan multiple trips, schedule activities, and assign meals — all saved locally in your browser.

## Features

- **Multiple trips** — create, switch between, and manage separate trips
- **Day-by-day planning** — each day has a title, an activity list, and lunch/dinner slots
- **Ideas pool** — stage activity ideas, then drag them onto any day or meal slot
- **Drag to reorder** — move items between days, within a day's list, or into meal slots
- **Persistent state** — trips are saved to `localStorage` and restored on reload

## Getting started

```bash
npm install
npm run dev
```

The app opens at `http://localhost:5173` with a sample Paris itinerary pre-loaded.

## Tech stack

React 19 · TypeScript · Vite
